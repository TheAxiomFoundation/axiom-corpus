601-1 C     SSI RECIPIENTS FROM CALIFORNIA

 

SSISupplemental Security Income recipients residing in California receive additional cash for food as part of their SSISupplemental Security Income state supplement.  Individuals receiving the SSISupplemental Security Income / SSPState Supplementary Payments benefits from California are not eligible for SNAPSupplemental Nutrition Assistance Program from Alaska in the same month.  The caseworker must ensure these benefits have stopped before issuing Alaska SNAPSupplemental Nutrition Assistance Program benefits.

 

Note:

Effective June 1, 2019, California will no longer be providing additional cash for food as part of their SSISupplemental Security Income state supplement.  For any months prior to June, the caseworker must ensure SSISupplemental Security Income state supplement payments were not received if Alaska SNAPSupplemental Nutrition Assistance Program eligibility is being determined for those months.
