605-3 D.      ELDERLY SIMPLIFIED APPLICATION PROJECT

 

This section describes policies related to Alaska's Elderly Simplified Application Project (ESAP). ESAPElderly Simplified Application Project Demonstration streamlines various requirements, such as verification and recertification, for eligible elderly/disabled households applying for or participating in SNAPSupplemental Nutrition Assistance Program. 

 

Background

Department of Health, Division of Public Assistance, requested and received federal approval to implement an Elderly Simplified Application Project (ESAP) demonstration project. This demonstration has been approved to operate effective 12/01/2024.

 

  1. Eligibility

    A household is eligible for ESAPElderly Simplified Application Project Demonstration if it does not receive countable earned income and where all adult members are: 

 

  • Age 60 or older; and/or 

  • Have a disability; 

 

Children may be present in the household, however the presence of a disabled child(ren) in the household alone will not make a household ESAPElderly Simplified Application Project Demonstration eligible.

 

  1. Application and Screening

    ESAPElderly Simplified Application Project Demonstration households will complete the same application as non-ESAP households for initial applications and recertifications. 

    ESAPElderly Simplified Application Project Demonstration households are eligible for Expedited Service. ESAP applications must be screened and processed to adhere to SNAPSupplemental Nutrition Assistance Program timeframes outlined in SNAP MS 601.

 

  1. Interview

    At application, ESAPElderly Simplified Application Project Demonstration households must complete an initial interview in order to determine eligibility for the ESAP demonstration project. This is done to help familiarize applicants with eligibility requirements, appropriate deductions, and reporting requirements.

    At recertification, if additional criteria is met, ESAPElderly Simplified Application Project Demonstration households may be eligible for a waived interview. See MS 605-3(D)(7). Even if the ESAP SNAPSupplemental Nutrition Assistance Program household is eligible for a waived interview, if ineligibility exists, never close an ESAP recertification without first attempting to schedule a recertification interview. 

 

  1. Verification

    ESAPElderly Simplified Application Project Demonstration households must comply with the verification requirements listed in SNAPSupplemental Nutrition Assistance Program MS 601-4. 

    The use of Data Systems and EISEligibility Information System Interfaces outlined in MS 601-4(A)(3) should be used as verification for ESAPElderly Simplified Application Project Demonstration households, whenever possible, to verify the following factors of eligibility: 

 

  • Gross nonexempt income,

  • Immigration status,

  • Social security number(s),

  • Identity, 

  • Disability, and

  • Child Support (paid)

When information is not available through available data exchanges, such as income from a pension, the household must provide documentation.

 

  1. Certification Period

    ESAPElderly Simplified Application Project Demonstration household will be certified for 36 months. 

 

  1. Reporting and Acting on Changes During the Certification Period

    ESAPElderly Simplified Application Project Demonstration households are subject to the same reporting requirements outlined inSNAPSupplemental Nutrition Assistance Program MS 604-4, however, ESAP participants are not required to submit Interim Reports. 

 

Note: 

See 605-3(D)(6) for Interim Report requirements when an ESAPElderly Simplified Application Project Demonstration household is converted to SNAPSupplemental Nutrition Assistance Program during the certification period. 

 

Unlike households certified for 24 months, ESAPElderly Simplified Application Project Demonstration households are waived from the requirement to be contacted once every 12 months. 

If a household no longer qualifies for ESAPElderly Simplified Application Project Demonstration, they should be immediately converted back to regular SNAPSupplemental Nutrition Assistance Program. See MS 605-3(D)(10). 

 

  1. Recertification

    ESAPElderly Simplified Application Project Demonstration household must complete and return the GEN 72Eligibility review application by the 15th day of the 36th month of certification in order to avoid any interruption in benefits. See SNAPSupplemental Nutrition Assistance Program MS 604-3(B) for additional information regarding recertification application filing deadlines. 

    The recertification interview is waived for ESAPElderly Simplified Application Project Demonstration households unless: 

  • The recertification application is not provided timely and verifications are not complete;

  • The household will be denied; 

  • Information provided is questionable, incomplete, or contradictory; or

  • The household requests an interview.

 

  1. Approval for ESAP

    When it is determined at application or recertification that a household is eligible for ESAPElderly Simplified Application Project Demonstration, the ESAP household must be notified of their ESAP status, reporting procedures, and responsibilities.

    The caseworker will send the SNAPSupplemental Nutrition Assistance Program ESAPElderly Simplified Application Project Demonstration Application Approved (F113) or SNAP ESAP Recertification Approved (F120) notice to these households. 

 

  1. Converting SNAP Households to ESAP

  1. When it is discovered that a regular SNAPSupplemental Nutrition Assistance Program household is eligible for ESAPElderly Simplified Application Project Demonstration at recertification, notify the household of their ESAP status, reporting procedures, and responsibilities. 

    The caseworker will send the SNAPSupplemental Nutrition Assistance Program ESAPElderly Simplified Application Project Demonstration Application Approved (F113) notice to these households. 

  2. When it is discovered that a regular SNAPSupplemental Nutrition Assistance Program household is eligible for ESAPElderly Simplified Application Project Demonstration during their certification period due to a reported change, the household will continue their current certification period and be converted to ESAP at their next recertification. 

 

  1. Converting Ineligible ESAP households to SNAP

    If an ESAPElderly Simplified Application Project Demonstration household becomes ineligible for ESAP during their certification period, but remains eligible for SNAPSupplemental Nutrition Assistance Program, the household is removed from the ESAP demonstration and converted to a regular SNAP household. 

    The caseworker will send the SNAPSupplemental Nutrition Assistance Program ESAPElderly Simplified Application Project Demonstration Household Converted to SNAP (F760) notice to these households. 

    EISEligibility Information System will automatically send the SNAPSupplemental Nutrition Assistance Program ESAPElderly Simplified Application Project Demonstration Household Converted to SNAP (X760) when EIS knows a household is no longer eligible for ESAP. 

 

Example: 

EISEligibility Information System will automatically send the X760 when a child, who is neither blind nor disabled, turns 18 and the household no longer meets ESAPElderly Simplified Application Project Demonstration eligibility. 

 

The household is still entitled to the 36-month certification period but will be required to submit the Interim Report (FSP 95SNAP Interim Report) in 6 month increments for the remainder of the 36-month certification period. 

At the end of the 36-month certification period, the household will be recertified according to normal SNAPSupplemental Nutrition Assistance Program recertification requirements if they remain ineligible for ESAPElderly Simplified Application Project Demonstration during that time.
