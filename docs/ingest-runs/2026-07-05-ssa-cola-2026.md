# US SSA COLA 2026 guidance ingest reasoning

Impact basis:

- rulespec-us issue 541 encodes 42 U.S.C. 415(i) COLA application with the
  2.8 percent 2026 cost-of-living adjustment (`us/policies/ssa/cola/2026`).
  The corpus already carries five 2026 SSA automatic-determination guidance
  blocks (contribution-and-benefit-base, quarter-of-coverage,
  retirement-earnings-test, pia-bend-points, substantial-gainful-activity)
  but not the COLA determination itself, so the 2.8 percent value had no
  verbatim grounding target.

Official sources:

- SSA Office of the Chief Actuary, Latest Cost-of-Living Adjustment page
  (https://www.ssa.gov/OACT/COLA/latestCOLA.html, page updated 2025-10-24).
  States: "The latest COLA is 2.8 percent for Social Security benefits and
  SSI payments," effective December 2025 and payable January 2026, with the
  CPI-W third-quarter computation (317.265 vs 308.729).
- Federal Register notice "Cost-of-Living Increase and Other Determinations
  for 2026", 90 FR 49047 (2025-11-03), FR Doc. 2025-19763, via the official
  GPO govinfo rendition. The formal 42 U.S.C. 415(i) determination; also
  carries the 2026 PIA bend points of $1,286 and $7,749.

Verification:

- Both official sources state 2.8 percent, matching the value encoded in
  rulespec-us issue 541. The Federal Register notice also confirms the 2026
  bend points ($1,286 / $7,749) that issue 541 cross-checks.

Scope:

- `us/guidance/ssa/cola/2026` (document) with `block-1` carrying the OACT
  determination text verbatim, following the citation-path pattern of the
  five existing 2026 SSA automatic-determination guidance blocks.
- `us/guidance/ssa/cola/2026/federal-register` (document) with `block-1`
  carrying the full Federal Register notice text.

Generated artifact:

- Manifest-driven extract-official-documents run; no corpus rows written by
  hand. The five existing SSA blocks predate the manifest convention (they
  were committed as provisions JSONL only); this run follows the current
  convention with manifest, archived sources, inventory, provisions,
  coverage, and a signed ingest manifest.
- The ssa.gov fetch uses the browser-impersonation request option because
  ssa.gov rejects non-browser TLS fingerprints.
- Output run id: 2026-07-05-ssa-cola-2026.
- Coverage result: complete; 4 source inventory rows matched 4 provision
  rows (2 documents + 2 blocks).

Command:

```bash
env -u UV_FROZEN uv run --extra dev axiom-corpus-ingest extract-official-documents \
  --base data/corpus \
  --version 2026-07-05-ssa-cola-2026 \
  --manifest manifests/us-ssa-cola-2026-official-documents.yaml
```
