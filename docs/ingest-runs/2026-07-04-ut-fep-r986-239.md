# 2026-07-04 Utah FEP R986-200-239 Focused Source

## Scope

Adds an additive section-level official corpus provision for Utah Administrative
Code R986-200-239, Determining the Amount of the Financial Assistance Payment.
This is the regulation section needed before encoding the remaining Utah FEP
gross-income, net-income, deduction, and payment-composition rules for
PolicyEngine parity.

The existing `2026-07-02-ut-fep-official-documents` corpus scope intentionally
remains unchanged because already-merged RuleSpec for R986-200-204 and
R986-200-230 is source-pinned to its whole-rule `block-1` provision. This run
adds a focused section provision rather than resegmenting that prior version.

## Command

```bash
uv run --extra dev axiom-corpus-ingest extract-official-documents \
  --base data/corpus \
  --version 2026-07-04-ut-fep-r986-239 \
  --manifest manifests/us-ut-fep-r986-239-official-document.yaml \
  --source-as-of 2026-07-04 \
  --expression-date 2026-07-04
```

## Expected Result

- Jurisdiction/document class: `us-ut` / `regulation`
- Source files: 1
- Focused provision: `us-ut/regulation/admin-rules/r986/200/239`
- Coverage: complete

The Utah Admin Rules HTML is generated from absolutely positioned spans. The
manifest therefore uses an anchor-range selector from the R986-200-239 heading
to the R986-200-240 heading, both inside official `awdiv` line containers.
