# Kansas State Supplemental Payment Program Source Ingest

This run supports the active PolicyEngine parity gap for `ks_sspp`.

Sources:

- K.S.A. 39-972 from the Kansas Office of Revisor of Statutes is the upstream statutory authority for the institutional supplemental income payment.
- KHPA Policy No. 2007-05-01 from the official KDHE Application Portal implements the State Supplemental Payment Program, including eligibility criteria and the monthly payment amount.

The existing Kansas KEESM corpus includes generic SSI and state supplement references, but it does not provide the SSPP-specific authority and implementation details needed for source-backed encoding.

Commands:

```bash
env -u UV_FROZEN uv run --extra dev axiom-corpus-ingest extract-official-documents \
  --base data/corpus \
  --version 2026-07-04-ks-sspp-statute \
  --manifest manifests/us-ks-sspp-statute-official-documents.yaml

env -u UV_FROZEN uv run --extra dev axiom-corpus-ingest extract-official-documents \
  --base data/corpus \
  --version 2026-07-04-ks-sspp-guidance \
  --manifest manifests/us-ks-sspp-guidance-official-documents.yaml
```
